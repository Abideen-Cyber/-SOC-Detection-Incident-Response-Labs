

SOC Detection & Incident Response Labs

This repository contains hands-on blue-team labs performed in a
VirtualBox environment to simulate real-world SOC detection scenarios.

## Tools Used
- Suricata IDS
- tcpdump
- auditd
- jq
- Apache
- Nmap, Gobuster, Nikto (attacker simulation)

## Labs Completed
- Day 1: Network Reconnaissance Detection (Nmap)
- Day 2: Web Enumeration Detection (Gobuster, Nikto)
- Day 3: Suspicious Login & Privilege Escalation Detection
- Day 4: SSH Brute-Force Detection & Authentication Log Analysis

## Skills Demonstrated
- Network traffic analysis
- IDS alert triage
- Incident classification
- Log analysis
- SOC documentation

> All attacks were performed in a controlled lab environment.
